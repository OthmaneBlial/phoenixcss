Welcome to PhoenixCSS

PhoenixCSS is a minimalistic CSS framework designed for semantic HTML and responsive design. It provides a set of utility classes, components, and a grid system to help you build modern, responsive websites quickly and efficiently. 

<link rel="stylesheet" href="../dist/css/phoenix.min.css" />
